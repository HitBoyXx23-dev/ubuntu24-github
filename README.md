# Ubuntu 24.04 for GitHub Codespaces

Run an Ubuntu 24.04 desktop and terminal directly inside GitHub Codespaces.

## Features

- Ubuntu 24.04

## Requirements

- GitHub Account
- GitHub Codespaces

## Setup

Open the repository in GitHub.

Click:

Code → Codespaces → Create codespace

Wait for the container to finish building.

Run:

```bash
sudo su
chmod +x start.sh
./start.sh
```
